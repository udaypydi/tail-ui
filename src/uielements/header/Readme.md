Default Header:

```js
<Header text="Default Header" />
```

Header Type:

```js
    <Header text="Header 1" />
    <Header text="Header 2" type="h2" />
    <Header text="Header 3" type="h3" />
    <Header text="Header 4" type="h4" />
```

Disabled Header:

```js
    <Header text="Header 1" disabled />
```
